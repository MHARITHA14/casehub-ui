CaseHub: Where Justice Meets Intelligence

CaseHub is an AI-powered legal management platform designed for judges and lawyers to streamline case handling, hearing scheduling, and judicial collaboration.

The system provides separate role-based dashboards for judges and lawyers.

Judge features include:
- Case scheduling and hearing management
- Previous hearing summaries
- Similar case search and document review
- Room booking for hearings
- Secure notes and task management
- Notifications and to-do tracking

Lawyer features include:
- Assigned case list
- Hearing schedules
- Document upload and evidence management
- Real-time updates and voice note uploads

Azure Services Used:
- Azure Static Web Apps for frontend deployment
- Azure App Service for backend hosting
- Azure SQL Database for structured data such as case details and room allocation
- Azure Blob Storage for unstructured data such as legal documents and evidence files
- Azure Cognitive Services for speech-to-text, text summarization, and document intelligence
- Azure Container Registry (ACR) for Docker image storage
- Azure Kubernetes Service (AKS) for container orchestration
- Terraform for infrastructure provisioning
- GitHub Actions / Azure DevOps for CI/CD automation

The goal is to improve efficiency, reduce manual work, and enable smarter judicial workflows using cloud and AI technologies.
