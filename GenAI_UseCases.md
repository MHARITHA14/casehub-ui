# CaseHub GenAI Use Cases

## Use Case 1: Hearing Transcription

Azure Service Used: Speech Service

Purpose:
Convert judge and lawyer hearing audio into text automatically using Speech-to-Text.

Why:
This reduces manual note-taking and improves documentation accuracy.

Example:
Court hearing voice → automatic text transcript


---

## Use Case 2: Case Summary Generation

Azure Service Used: Language Service

Purpose:
Generate short summaries of long legal documents and previous hearings.

Why:
Helps judges quickly understand important case details.

Example:
Large case document → short legal summary


---

## Use Case 3: Similar Case Search

Azure Service Used: Document Intelligence + Language Service

Purpose:
Analyze legal documents and identify similar past cases.

Why:
Helps judges refer previous judgments for faster decision making.

Example:
Current theft case → similar old theft case + judgment
